Redeem your Azure Pass code

  1. Log in to Azure Portal with the new user account.

  2. There should be no subscriptions.

  3. Go to https://www.microsoftazurepass.com, log in with the new user account.

  4. Redeem your Azure Pass Code.

  5. At the end there should be an Azure Pass - Sponsorship subscription associated with the Azure AD tenant.

Congratulation! You redeemed your Azure Pass code.
