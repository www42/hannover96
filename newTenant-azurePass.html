Create a new tenant and redeem your Azure Pass code

Important! Telephone number is required.

  1. Go to https://account.azure.com/organization

  2. Fill in the form. Telephone number has to correct! Remember your user account (e.g. Albert@hamburg.onmicrosoft.com) and your password!

  3. After completion you will be offered an Azure Free Account - Don't use it! Abort!

  4. Log in to Azure Portal with the new user account. There should be no subscription.

  5. Go to https://www.microsoftazurepass.com, log in with the new user account, and redeem your Azure Pass Code.

  6. At the end there should be an Azure Pass - Sponsorship subscription associated with the tenant.

Congratulation! You have created a new Azure AD tenant associated with an Azure Pass - Sponsorship subscription.
